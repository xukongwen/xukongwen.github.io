# 般若波羅蜜多心經




觀自在菩薩。<br>
行深般若波羅蜜多時。<br>
照見五蘊皆空。度一切苦厄。<br>
舍利子。<br>
色不異空。空不異色。<br>
色即是空。空即是色。<br>
受想行識。亦復如是。<br>
舍利子。<br>
是諸法空相。<br>
不生不滅。不垢不淨。不增不減。<br>
是故空中無色。<br>
無受想行識。<br>
無眼耳鼻舌身意。<br>
無色聲香味觸法。<br>
無眼界。<br>
乃至無意識界。<br>
無無明。<br>
亦無無明盡。<br>
乃至無老死。<br>
亦無老死盡。<br>
無苦集滅道。<br>
無智亦無得。<br>
以無所得故。<br>
菩提薩埵。<br>
依般若波羅蜜多故。<br>
心無罣礙。無罣礙故。無有恐怖。<br>
遠離顛倒夢想。究竟涅槃。<br>
三世諸佛。依般若波羅蜜多故。<br>
得阿耨多羅三藐三菩提。<br>
故知般若波羅蜜多。<br>
是大神咒。是大明咒。是無上咒。是無等等咒。<br>
能除一切苦。真實不虛。<br>
故說般若波羅蜜多咒。<br>
即說咒曰。<br>
揭諦揭諦　波羅揭諦　波羅僧揭諦　菩提薩婆訶<br>
