# 如何使用

![](https://s3.ax1x.com/2021/02/15/y6QCIs.png)

## 回家

none-os有一个类似主页的东西，或者叫“家”。

任何时候找不到北了，就敲击<空格-m>

不是同时按下，而是敲击空格后，立刻敲击m。

## 超级寻找

<空格-f>

这时你输入任何字符，会搜寻当下目录下的所有文件。
支持模糊搜寻。

这是寻找某个文件的方法之一。

如果这个方法不能找到你的文件可以

<空格-e>

这里可以控制光标到某个具体目录寻找你要的文件

## 帮助

<空格-h>，会带你进入编辑好的帮助说明

## 保存文件

<空格-s>

## 退出none-os

<空格-q>

# 简易的Vim教学

## vim大约有两种模式

1. 观赏与控制模式
2. 文字编辑模式

观赏模式指你可以利用方向键浏览整个文档，在文档中游走。
同时你可以做删除，拷贝，黏贴等等动作。
进入这个模式按
ESC

当你想输入具体文字时
一版按
i
当然也可以是
o
更具体的在后面会有说明
你想退出了那么就按ESC
