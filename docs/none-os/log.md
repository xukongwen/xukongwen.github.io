今天无意间购买了这个域名，把之前kongwen.one这个网站直接换成了none-os.com。
简单修改了网站内容，算是第一版。

现在只有各种意淫还没有什么实质上的东西。

未来希望慢慢把可以放出下载的东西放出来。

现在大概的进度我觉得还不错啊，虽然我用了两周时间来制作系统，几乎从零开始学习Linux和Emacs，但是进益颇多。

系统上是完全可用了，开机屏蔽掉所有信息，直接进入Emcas，安装好了中文显示和输入，以及一些基本Package。

最重要的是这个Writeroom插件，至少是我写作时不可缺少的。

用到的Package

- Writeroom （文字居中）

- Pyim （中文输入）

- Neotree （文件树）

- Magit （Git神器）

- Ivy （命令自动补全）

基本上就这些，未来也不希望弄一大堆，因为要满足单纯写作并不是特别困难。

现在做的配置基本上是给居中写作一个尽可能清创的显示。
