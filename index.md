# 何为None-OS

首先这是一个自娱自乐的意淫项目，请大家不要当真。

None-OS是一个开机即进入一个干净简单写作模式的低功耗硬件和软件的集合工具。
{: .label .label-yellow .fs-7 }

具体说来，她是由一个树莓派ZREO W，机械硬盘，显示器组成的低功耗电脑，里面运行的系统是定制好的Linux+Emacs。（开机即用）

她几乎只服务一个目的：纯粹快乐的书写文本。

她大约面向类似我这种人，既是，不会Linux，更不会Emacs，但需要一款顺手的文字编辑器。

这款工具并非面对程序员，而是，有写写小说，写写文章之类的人。

我的需求很简单：

极低功耗，极简单和有限的功能，速度快，流畅，干净，可方便自定义。

现在这个版本是运行在纯文字（TTY）模式下的Linux，排除一切干扰，只专注于写作本身。

我不知道世界上是否有同样需求的人，愿意在纯文本模式下工作。

如果有，这个网站可以提供给你一些参考。
