# 何为None-OS

None-OS是一个为树莓派低功耗电脑制作的简单写作工具（系统镜像）。

你刷入镜像，开机，就直接进入一个配置好中文写作环境的Vim。

这个环境里没有任何现代图形系统，只有简陋的文字菜单，同时不支持鼠标操作。

# 不能做到的

- 无法浏览网页

- 无法玩游戏

- 无法社交媒体

- 无法收发邮件

- 无法看视频

- 无法看图片

- 无法欣赏广告

- 基本上，她啥也做不了

# 能做到的

- 专注的写文字（小说，诗剧，剧本，论文，网页，程序，笔记，日记，等等）

- 将写好的文字传输到别的设备或者网路

# 给谁用

- 我自己。。。

- 非程序员，喜爱无干扰写作环境，不太懂Linux以及Vim（我），喜欢低功耗电脑的人。。。

![](https://s3.ax1x.com/2021/02/14/yywEIf.png)


实际上这个东西既是我现在使用的写作工具，也是我正在写作小说里的一个“科幻”产物。

未来人类因为能源紧缺，进入严苛的能源配给时代。

那时的人们不再像我们随意挥霍着碳排放，手机，电脑的功耗被严格限制。

于是产生了这么一个非常局限的，一切基于纯文本的生态系统。

同时那时的人们对超级互联网企业有很大的恐惧，更倾向于一种线下的（off the line）的生活模式，那么这个系统也对线下的数据保存和数据交流有较好的支持。

我个人也不知道未来人类会走到何处，但是我对现在超级大公司引领我们的生活方式报以怀疑态度。

所以这个小项目也是个人的一个文化试验，看看在低功耗，简单的数码设备中生活，会有什么样的体验。

我喜欢中国古道家描述的生活，用现在的话来说就是低功耗而简单的生活，貌似这样做并不会降低幸福度，反而会让人在更持久和稳定的恬静中度过。

我并不确定这个选择是好是坏，那么就试验一下吧。
