# 何为None-OS

None-OS是一个为树莓派低功耗电脑制作的简单写作工具（系统镜像）。

你刷入[镜像](/docs/download)，开机，就直接进入一个配置好中文写作环境的Vim。

这个环境里没有任何现代图形系统，只有简陋的文字菜单，同时不支持鼠标操作。

她是一个非常低能的工具，所以叫None-OS（无能的系统）

# 不能做到的

- 无法浏览网页

- 无法玩游戏

- 无法社交媒体

- 无法收发邮件

- 无法看视频

- 无法看图片

- 无法欣赏广告

# 能做到的

- 专注的写文字（小说，诗剧，剧本，论文，网页，程序，笔记，日记...）

- 将写好的文字传输到别的设备或者网路

# 为何低功耗

这个低能的小项目始于2021年春节左右。

这个冬天北京的雾霾又比较严重，上面的照片就拍摄于大年初一。

我相信雾霾终会离去，但并非因为临时的治理，或者依附于某种高科技。

她的发生只能依赖于人们生活态度的改变。

最近几年我在尝试以各种方式过所谓低碳简单的生活，这个None-OS是在电脑生活上的尝试。

我并不确定将我80%的工作转移到一个只有0.5v功耗的小电脑上能对雾霾有任何作用，但是至少在我自己使用的这一个月来，感觉不错。我可以暂时逃离无止尽的广告，可以安心的写一会字。

所以这个小项目也是个人的一个文化试验，看看在低功耗，简单的数码设备中生活，会有什么样的体验。

最近几年喜欢中国道家的无为生活态度，能简单就简单，能不干就不干。虽然被世人嘲笑和诟病，但是我愿意尝试一下，看看到底是喜是悲。

我现在正用这个设备写这个网站，同时在写一本送给女儿的书，以及一个小说。

![](https://s3.ax1x.com/2021/02/14/yy0Fl4.jpg)

这个小说里面，人们因为能源危机，被迫在较为极端的能源供给下生活。比如被迫每个只能用5v以内的电子设备。

人们回归到电脑创造初期的文字操作系统，未来科技非但没有进展，反而出现了反古。

小说就描写了那个时候某些人的一些琐事。
