这里简单介绍文章的发布流程！

发布一篇文章时，需要在 `_post` 目录下简历一个 markdown 文件，文件名以日期加一个英文的标题，这个标题会作为文章的 url，文章的内容按标准的 markdown 格式编写，**但是文章开头必须加入如下的头：**

```
---
layout: post
title:  "文章发表示例"
author: "Xiu Kuang"
---
```

除了 author 字段外，另两个字段都是必须的。

可以在 `_config.yml` 文件中修改一些配置。

`_pages/about.md` 是一个自我介绍页面，可以进行修改。

如果需要插入图片，可以放到 assets/imgs/ 目录下，然后插入：

![test image](/assets/imgs/test.jpg)

也可以直接使用外部链接：

![test external image](http://xxx.com/xxx.jpg)

如果需要插入 js，可以把 js 文件放到 assets/js 目录下，然后直接将如下内容插入到文章中：

```
<script src="/assets/js/test.js" type="text/javascript"></script>
```

`_posts` 目录下的文章是一些示例文章，后续可将其删除。
